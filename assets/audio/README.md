# Audio Placeholder

Put your first music track here as one of:

```text
intro_battle.ogg
intro_battle.mp3
intro_battle.wav
```

The game will run without it, using only the battle timer. This keeps the code testable before the final song is ready.

For menu/dialogue/result background music, use:

```text
background_music.ogg
background_music.mp3
background_music.wav
```

`forbackroundmusic.mp4` was converted to `background_music.ogg` because SFML handles OGG more reliably than MP4.
