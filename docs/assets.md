# Asset List

## Required For Vertical Slice

- `assets/charts/intro_battle.chart`: bundled text chart.
- `assets/audio/intro_battle.mp3`: first music track. The game also accepts `intro_battle.ogg` or `intro_battle.wav`.
- `assets/audio/background_music.ogg`: looped music for menu, dialogue, and result screens.
- Yumi portraits in `assets/charts/yumi`: normal, angry, happily angry, and pointing finger. `DialogueView` falls back to a drawn placeholder if a file is missing.
- UI sprites: currently drawn from SFML rectangles/text for a chunky Flash/DS look.

## Later Assets

- custom UI frames
- additional `.chart` files
- additional `.ogg` tracks
- result screen illustrations
